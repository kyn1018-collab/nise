===== 나이스(NEIS) 교무업무 처리 안내 =====
※ 이 자료는 초등학교 나이스 교무업무 처리 절차 안내입니다.

----- 1. 출결 관리 (출석부 처리) -----

[메뉴 경로]
나이스 로그인 → 교무업무 → 출결관리 → 일별출결관리

[처리 절차]
1. 나이스(https://neis.go.kr) 에 접속하여 공인인증서 또는 간편인증으로 로그인합니다.
2. 상단 메뉴에서 [교무업무]를 클릭합니다.
3. 좌측 메뉴에서 [출결관리] → [일별출결관리]를 선택합니다.
4. 조회 조건에서 학년, 반, 날짜를 선택하고 [조회] 버튼을 클릭합니다.
5. 학생 목록에서 결석/지각/조퇴/결과(병결, 무단, 기타 등)를 선택하여 입력합니다.
6. 결석 사유는 해당 항목 옆 비고란에 입력합니다.
7. 입력 완료 후 [저장] 버튼을 클릭합니다.

[출결 종류 및 코드]
- 결석: 무단결석, 병결, 기타결석(가정사정 등)
- 지각: 무단지각, 병(의료기관), 기타지각
- 조퇴: 무단조퇴, 병(의료기관), 기타조퇴
- 결과: 무단결과, 병(의료기관), 기타결과

[유의사항]
- 질병결석은 의사 진단서 또는 학부모 의견서 첨부 필요
- 무단결석 3일 이상 시 교육지원청 보고 의무
- 매월 말일 이전 해당 월 출결 마감 처리

----- 2. 성적 처리 -----

[메뉴 경로]
나이스 로그인 → 교무업무 → 성적관리 → 단원/수행평가입력 (또는 학기말 성적 처리)

[단원/수행평가 점수 입력 절차]
1. [교무업무] → [성적관리] → [교과학습 발달상황] → [단원평가 점수입력]을 선택합니다.
2. 학년/반/교과를 선택하고 [조회]를 클릭합니다.
3. 학생별 점수를 입력합니다.
4. 입력 완료 후 [저장]을 클릭합니다.

[학기말 성적(통지표) 처리 절차]
1. [교무업무] → [성적관리] → [교과학습발달상황] → [성적일괄입력]을 선택합니다.
2. 학년/반/학기를 선택합니다.
3. 각 교과의 성취도(수/우/미/양/가 또는 A/B/C) 및 원점수를 입력합니다.
4. 과목별 특기사항을 입력합니다 (선택사항).
5. 입력 완료 후 [저장]을 클릭합니다.

[통지표 출력]
1. [교무업무] → [성적관리] → [통지표출력]을 선택합니다.
2. 학년/반/학기를 선택하고 [조회]합니다.
3. 전체 또는 개별 학생을 선택하고 [출력]을 클릭합니다.

[성적 마감 절차]
1. 담임교사: 성적 입력 완료 후 [성적마감신청] 클릭
2. 교무부장: [성적마감승인] 메뉴에서 학급별 성적 확인 후 승인
3. 교감/교장: 최종 결재

----- 3. 학적 관리 -----

[메뉴 경로]
나이스 로그인 → 교무업무 → 학적관리

[전입학생 처리 절차]
1. [교무업무] → [학적관리] → [전입학처리]를 선택합니다.
2. [전입학생조회] 탭에서 전입학생 정보를 확인합니다.
3. 해당 학생의 배정 학년/반을 지정합니다.
4. [저장]을 클릭합니다.

[전출학생 처리 절차]
1. [교무업무] → [학적관리] → [전출처리]를 선택합니다.
2. 전출학생을 검색합니다.
3. 전출일, 전출 사유, 전출 학교를 입력합니다.
4. [저장]을 클릭합니다.

[학생 기본정보 수정]
1. [교무업무] → [학적관리] → [학생정보관리]를 선택합니다.
2. 수정할 학생을 검색합니다.
3. 해당 정보(주소, 연락처, 보호자 정보 등)를 수정합니다.
4. [저장]을 클릭합니다.

----- 4. 창의적 체험활동 입력 -----

[메뉴 경로]
나이스 로그인 → 교무업무 → 창의적체험활동 → 영역별 입력

[자율활동 입력]
1. [교무업무] → [창의적체험활동] → [자율활동] → [자율활동입력]을 선택합니다.
2. 학년/반을 선택하고 [조회]합니다.
3. 학생별 자율활동 내용(시간 및 특기사항)을 입력합니다.
4. [저장]을 클릭합니다.

[동아리활동 입력]
1. [교무업무] → [창의적체험활동] → [동아리활동] → [동아리활동입력]을 선택합니다.
2. 동아리명, 활동 시간, 특기사항을 입력합니다.
3. [저장]을 클릭합니다.

[봉사활동 입력]
1. [교무업무] → [창의적체험활동] → [봉사활동] → [봉사활동입력]을 선택합니다.
2. 봉사활동 기관, 일자, 시간, 내용을 입력합니다.
3. [저장]을 클릭합니다.

[진로활동 입력]
1. [교무업무] → [창의적체험활동] → [진로활동] → [진로활동입력]을 선택합니다.
2. 진로활동 내용 및 특기사항을 입력합니다.
3. [저장]을 클릭합니다.

----- 5. 학교생활기록부(학생부) 관리 -----

[메뉴 경로]
나이스 로그인 → 교무업무 → 학교생활기록부

[학생부 항목별 입력 경로]
- 인적사항: [학교생활기록부] → [학생정보] → [인적사항]
- 학적사항: [학교생활기록부] → [학생정보] → [학적사항]
- 출결상황: [학교생활기록부] → [출결상황] (출결관리에서 자동 연동)
- 교과학습발달상황: [학교생활기록부] → [교과학습발달상황]
- 창의적체험활동: [학교생활기록부] → [창의적체험활동상황]
- 행동특성 및 종합의견: [학교생활기록부] → [행동특성및종합의견]

[행동특성 및 종합의견 입력 절차]
1. [교무업무] → [학교생활기록부] → [행동특성및종합의견]을 선택합니다.
2. 학년/반을 선택하고 [조회]합니다.
3. 학생별 종합의견을 입력합니다 (500자 이내).
4. [저장]을 클릭합니다.

[학생부 출력]
1. [교무업무] → [학교생활기록부] → [학생부출력]을 선택합니다.
2. 학년/반/학생을 선택합니다.
3. [출력]을 클릭합니다.

----- 6. 교원 복무 관리 -----

[연가/조퇴/외출 신청 절차]
1. 나이스에 로그인합니다.
2. [행정업무] → [복무관리] → [근태관리] → [연가/결근/지각관리]를 선택합니다.
3. [신청] 탭에서 연가 종류(연가, 병가, 공가, 특별휴가 등)를 선택합니다.
4. 신청 기간, 사유를 입력합니다.
5. [저장] 후 [신청] 버튼을 클릭합니다.
6. 결재권자(교감 → 교장)의 승인을 기다립니다.

[출장 신청]
1. [행정업무] → [복무관리] → [출장관리]를 선택합니다.
2. 출장 목적, 일시, 장소를 입력합니다.
3. 저장 후 결재 상신합니다.

[연수 이수 등록]
1. [행정업무] → [연수관리] → [연수이수현황]을 선택합니다.
2. 이수한 연수의 기관, 과정명, 시간, 이수일을 입력합니다.
3. [저장]을 클릭합니다.

----- 7. 보건/급식 관련 -----

[학생 건강 정보 입력]
1. [교무업무] → [보건관리] → [학생건강정보]를 선택합니다.
2. 신장, 체중 등 건강검사 결과를 입력합니다.

[급식 현황 조회]
1. [행정업무] → [급식관리] → [급식현황]을 선택합니다.

----- 8. 나이스 공통 사용 팁 -----

[로그인 방법]
- PC: https://neis.go.kr → 교직원 로그인 → 공인인증서 또는 간편인증(PASS 앱 등)
- 모바일: 나이스 모바일 앱 설치 후 로그인

[자주 발생하는 오류 해결]
- 조회 안 될 때: 학년도/학기 설정 확인
- 저장 안 될 때: 마감 여부 확인, 권한 확인
- 로그인 오류: 공인인증서 갱신 또는 간편인증 재등록
