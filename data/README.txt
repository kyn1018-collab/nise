===== 나이스 도우미 - 참고자료 추가 방법 안내 =====

이 폴더(data/)에 나이스 관련 참고자료를 추가하면 챗봇이 해당 자료를 기반으로 답변합니다.

[지원 파일 형식]
- .txt (텍스트 파일) ← 가장 권장
- .md (마크다운 파일)

[PDF 참고자료 추가 방법]
1. PDF 파일을 열어 내용을 복사합니다 (Ctrl+A → Ctrl+C)
2. 메모장을 열어 붙여넣기 합니다 (Ctrl+V)
3. '파일명.txt' 로 저장합니다 (예: 나이스_성적처리_매뉴얼.txt)
4. 저장한 .txt 파일을 이 폴더에 복사합니다.
5. index.html 파일의 DATA_FILES 배열에 파일명을 추가합니다.

[DATA_FILES 배열 수정 방법]
index.html 파일을 열어 아래 부분을 찾아 파일명을 추가합니다:

const DATA_FILES = [
  'data/나이스_기본업무_안내.txt',
  'data/추가파일명.txt',   ← 여기에 추가
];

[현재 등록된 파일]
- 나이스_기본업무_안내.txt : 출결/성적/학적/창체/학생부/복무 처리 절차
